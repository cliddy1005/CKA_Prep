# NetForge Lab

Static network engineering training app for GitHub Pages.

## Included
- Multi-file structure
- Learning modules
- CLI-style troubleshooting simulator
- Validation with local score tracking
- Quick reference
- PWA manifest + service worker
- iPhone home screen friendly metadata

## Deploy to GitHub Pages
1. Upload all files in this folder to the root of your repository.
2. Make sure `index.html` stays at the repo root.
3. In GitHub, enable Pages for the repository branch.
4. Open the published URL on iPhone Safari and choose **Add to Home Screen**.

## Structure
- `index.html`
- `manifest-net.json`
- `sw.js`
- `assets/css/styles.css`
- `assets/js/data.js`
- `assets/js/ui.js`
- `assets/js/app.js`
- `assets/icons/*`
